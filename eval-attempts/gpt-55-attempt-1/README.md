# Przeprogramowani Modern Site

Statyczna strona Astro + React + Tailwind przygotowana pod Cloudflare Pages.

## Komendy

```bash
npm install
npm run dev
npm run build
npm run preview
```

Cloudflare Pages:

- build command: `npm run build`
- output directory: `dist`

`wrangler.toml` zawiera `pages_build_output_dir = "./dist"`.

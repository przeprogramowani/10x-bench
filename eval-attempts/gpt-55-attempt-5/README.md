# Przeprogramowani.pl

Nowoczesna, responsywna strona projektu Przeprogramowani.pl zbudowana w Astro, React i Tailwind CSS.

## Komendy

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Cloudflare Pages

Build command:

```bash
npm run build
```

Output directory:

```bash
dist
```

Konfiguracja `wrangler.toml` zawiera `pages_build_output_dir = "dist"` oraz aktualną datę kompatybilności.

## Źródła treści

Snapshot treści w `src/data/site.ts` powstał na podstawie publicznych stron Przeprogramowanych, kursów, Spotify for Creators i RSS kanału YouTube.
